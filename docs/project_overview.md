# 시각 미감 AI 연구

## 1. 프로젝트 개요

본 프로젝트는 이미지를 입력으로 받아 미적 품질을 평가하고, 그 결과를 사용자가 이해할 수 있는 피드백으로 제공하는 AI 시스템을 개발하는 것을 목표로 한다.

초기 연구는 착장 사진의 미적 평가를 중심으로 진행한다. 이후에는 장소 기반 코디 추천 웹앱, UI 디자인 평가, PPT 슬라이드 평가, 인테리어 이미지 피드백 등 다양한 시각 결과물 평가 영역으로 확장할 수 있다.

본 프로젝트는 단순히 이미지를 분류하는 것이 아니라, 시각적 결과물에 대한 미감 평가와 설명 가능한 피드백 생성을 함께 다루는 것을 목표로 한다.

---

## 2. 핵심 연구 방향: Aesthetic Scoring & Feedback

본 프로젝트의 1차 목표는 착장 사진을 대상으로 한 미적 점수화 및 피드백 AI를 구현하는 것이다.

입력 이미지를 기반으로 미적 점수 또는 등급을 예측하고, 구도·색감·밝기·균형·배경 복잡도 등을 기준으로 설명 가능한 피드백을 제공한다.

### 입력

- 착장 사진
- 전신 또는 반신 이미지
- 단일 인물 중심 이미지

### 출력

- 미적 점수 또는 등급
- 설명 가능한 피드백 문장

예시 출력:

```text
Score: 7.2 / 10

Feedback:
전체적인 구도는 안정적입니다. 피사체가 중앙에 잘 배치되어 있고, 착장의 색상도 배경과 적절히 분리됩니다. 다만 배경 요소가 다소 많아 시선이 분산될 수 있습니다.
```

---

## 3. 데이터셋: AVA

본 연구는 AVA(Aesthetic Visual Analysis) 데이터셋을 기반으로 시작한다.

AVA는 이미지의 미적 품질 평가를 위해 사용되는 대표적인 데이터셋이며, 각 이미지에 대해 1~10점 평가 분포를 제공한다. 단일 정답이 아니라 여러 평가자의 점수 분포를 포함하므로, 주관적인 미적 판단을 통계적으로 다룰 수 있다.

초기 실험에서는 전체 데이터를 모두 사용하지 않고, 재현 가능한 subset을 생성하여 사용한다.

### 초기 데이터 전략

- 전체 AVA 데이터 중 일부 subset 사용
- seed 고정
- 점수 분포를 고려한 sampling
- train / validation / test split 고정
- Low / Medium / High 등급 분류부터 시작

---

## 4. 모델 설계 방향

초기 모델은 경량 CNN 기반 baseline으로 구성한다.

목표는 단순히 큰 모델을 사용하는 것이 아니라, CPU 환경과 Raspberry Pi 적용 가능성을 고려한 실현 가능한 모델을 설계하는 것이다.

### 후보 모델

- ResNet-18
- MobileNet 계열
- EfficientNet-B0

### 출력 방식

초기 실험에서는 점수 회귀보다 등급 분류를 우선 고려한다.

- Low
- Medium
- High

이후 실험이 안정화되면 1~10점 회귀 또는 점수 분포 예측으로 확장할 수 있다.

---

## 5. 설명 가능한 피드백 설계

본 프로젝트는 단순히 점수를 예측하는 것에서 끝나지 않고, 사용자가 이해할 수 있는 피드백을 제공하는 것을 목표로 한다.

피드백은 모델의 예측 결과와 이미지 기반 시각 지표를 함께 활용하여 생성한다.

### 피드백 기준 후보

- 구도 안정성
- 피사체 위치
- 색 대비
- 밝기 및 노출
- 선명도
- 배경 복잡도
- 착장과 배경의 분리도
- 전체 균형감

초기에는 LLM 기반 생성이 아니라, 규칙 기반 템플릿 피드백으로 구현한다.

## 6. Raspberry Pi 적용 가능성

본 프로젝트는 Raspberry Pi + 카메라 기반 착장 평가 장치로 확장할 수 있다.

라즈베리파이에서는 고해상도 실시간 분석보다는, 촬영된 이미지를 리사이즈한 뒤 경량 모델로 추론하는 방식을 사용한다.

### 예상 흐름

```text
카메라 촬영 → 이미지 전처리 → 경량 모델 추론 → 점수/간단 피드백 출력
```

### 적용 시나리오

- 착장 평가 키오스크
- 스마트 거울
- 전시용 AI 피드백 장치
- 오프라인 이미지 평가 시스템

## 7. 서비스 확장: Scene-to-Outfit

Aesthetic Scoring & Feedback 연구가 안정화되면, 이를 기반으로 장소 기반 코디 추천 웹앱으로 확장할 수 있다.

### Scene-to-Outfit 개념

장소 사진이나 레퍼런스 이미지를 입력하면, 해당 분위기에 어울리는 코디 컨셉을 추천하는 서비스이다.

초기 버전에서는 사용자의 옷장 DB 없이 장소 분위기에 맞는 코디 컨셉을 추천하고, 이후에는 개인 옷장 DB와 연결하여 실제 보유 아이템 기반 추천으로 확장할 수 있다.

### 단계별 확장

1. 장소 사진 기반 코디 컨셉 추천
2. 개인 옷장 등록 기능
3. 보유 아이템 기반 코디 추천
4. 부족한 아이템 보조 추천
5. 미적 피드백 엔진을 활용한 코디 적합도 평가

## 8. 장기 확장 아이디어

본 프로젝트의 핵심은 시각 결과물에 대한 미적 평가와 설명 가능한 피드백이다.

따라서 착장 평가 외에도 다양한 분야로 확장할 수 있다.

### 확장 후보

- UI 디자인 평가
- PPT 슬라이드 평가
- 인테리어 이미지 피드백
- 코디 스타일 평가
- 사진 구도 피드백

각 도메인마다 평가 기준은 다르지만, 공통적으로 이미지를 입력으로 받아 미적 품질을 평가하고 개선 방향을 제안한다는 점에서 동일한 구조를 가진다.
